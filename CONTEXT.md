# Portal Messenger

Portal Messenger is a playful realtime teaching application presented through the visual language of early-2000s desktop software.

## Language

**Portal Messenger**: The public-facing chat application, formally presented as "Portal Messenger: Corporate Edition."

**Portal Systems**: The fictional software company whose workplace contains the Shared Public Office.

**Shared Public Office**: The single communal workspace where New Hires participate together.

**New Hire**: A person authenticated by Clerk and participating in the Shared Public Office.

**New Hire Profile**: The current public name and picture supplied by Clerk as short-lived Portal identity claims.

**Office Channel**: One of the fixed, standard conversation spaces in the Shared Public Office. General uses the Portal channel ID `general`; Announcements uses `announcements-v2` after the production migration recorded in ADR 0011. Both channels have the same Portal properties and differ only in their application-facing identity and copy.
